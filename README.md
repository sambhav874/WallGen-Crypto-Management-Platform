# WallGen :: Crypto Management Platform

Welcome to the WallGen, a decentralized application (dApp) built to enhance your Solana blockchain experience. This platform is designed to provide a range of functionalities including wallet management, token creation, transaction exploration, and more.

## Features

- **Donate:** Contribute SOL to support the project.
- **Generate Wallets:** Create and manage Solana wallets.
- **Transactions:** View and manage your transactions.
- **Contact:** Get in touch for support or inquiries.
- **Explorer:** Explore various blockchain transactions.
- **Request Airdrop:** Request SOL airdrops to your wallet.
- **Create Token:** Mint new tokens on the Solana blockchain.
- **Fetch Token Metadata:** Retrieve metadata for tokens created or managed on Solana.

## Tech Stack

- **Frontend:** Next.js, TypeScript, Tailwind CSS
- **Blockchain:** Solana
- **File Storage:** Pinata
- **Token Management:** Metaplex
- **Blockchain Data:** Alchemy
- **Form Handling:** Formspree

## Getting Started

### Prerequisites

- Node.js and npm installed
- Solana wallet and network setup

### Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/sambhav874/WallGen-Crypto-Management-Platform
    ```

2. Navigate to the project directory:
    ```bash
    cd crypto_wallet
    ```

3. Install dependencies:
    ```bash
    npm install
    ```

4. Create a `.env.local` file in the root directory and configure it with your environment variables (e.g., API keys).

5. Start the development server:
    ```bash
    npm run dev
    ```

### Usage

- Open `http://localhost:3000` in your browser to access the application.
- Use the navigation to explore functionalities such as donating, creating wallets, viewing transactions, and more.
